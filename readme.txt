== Description ==

Category Post List Widget is a light widget which displays the most recent posts from a certain category.
Features:
1. Set the category from which the posts should come.
2. Set Widget dimensions to display post of selected category on front end.
3. Set number of posts to show.
4. Option to change ordering of posts.
5. Option to change sort type of posts (can set sort by date, title, number of comments, random).
6. Option to select different effects. 
7. Option to show the post excerpt and how long the excerpt should be.
8. Option to select different display options.
9. Set Thumbnail size.
10. Set different date format.

== Installation ==

1. Upload category-post-list-widget.zip to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto the Widget tab of the Presentation section and configure the widget.

== Frequently asked questions ==

= A question that someone might have =

An answer to that question.

1. How do i show post of particular category using this plugin?
a) Find 'Appearance' menu in your WordPress admin panel. Click on "widget" sub-menu.
b) Drag & Drop 'Category Post List' widget in Widget area.
c) Configure the different widget options.

2. Can I change date format?
Yes, you can change date format. You can use any PHP standard date format e.g. F j,y for November 28, 2012.

3. Can i change different effects?
Yes, you can change effects of displaying your post of particular category. Effect option is provided by this category.
